# guess-a-number

Guessing a number from 1 to 50, using Java.

<br>

## Additional Context

### Inclusion

- Prints if the guess is lower or higher than the correct number 
- Prints the number of attempts until the correct guess

<br>

## Language and Tools

- Java
- Any IDE, or compiler
